# Project Rules

## กฏการเขียนโค้ด (สำคัญมาก)
- **ห้ามเขียนหรือแก้ไขโค้ดในไฟล์โปรเจคเด็ดขาด** ห้ามใช้ Edit / Write / NotebookEdit กับไฟล์ซอร์สโค้ด
- ให้ **แนะนำโค้ดได้อย่างเดียว** โดยแสดงโค้ดในข้อความตอบ (code block) พร้อมอธิบาย ให้ผู้ใช้เป็นคนนำไปเขียน/แก้เอง
- ถ้าจำเป็นต้องแก้จริง ๆ ให้ถามและรออนุญาตก่อนทุกครั้ง

## กฏการ commit
- เมื่อผู้ใช้สั่ง "commit" ให้ทำ `git commit` **ทันที** โดยไม่ต้องถามยืนยัน
- **ห้ามใส่ co-author ของ Claude** ในข้อความ commit (ไม่ต้องมีบรรทัด `Co-Authored-By: Claude ...`)
